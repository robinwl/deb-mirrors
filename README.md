# robinwl/deb-mirrors

[nginx] with [ngx_http_geoip_module] that redirect [apt-get] requests to closest Debian or Ubuntu mirror.

Read more on [http://mirror.naas.io].

### Development

Uses [Docker], [fig] and [Vagrant] as devstack. 

Want to contribute? Great!
